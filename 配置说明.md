# 配置说明

## 首次使用配置

### 1. 配置 Planner

复制配置文件模板：

```bash
cd planner
cp config.toml.example config.toml
```

编辑 `config.toml`，填入你的 API 密钥和模型 ID：

```toml
[models.'doubao_1_5_ui_tars_250328']
name = "YOUR_MODEL_ID_HERE"          # 替换为你的模型 ID
api_key = "YOUR_API_KEY_HERE"        # 替换为你的 API Key
```

### 2. 获取 API 密钥

1. 访问 [火山引擎控制台](https://console.volcengine.com/ark)
2. 创建或选择一个模型
3. 获取 API Key 和 Model ID
4. 将它们填入 `planner/config.toml`

### 3. 启动服务

```bash
./start-all.sh
```

## 安全提示

⚠️ **重要**: 
- `config.toml` 文件已被添加到 `.gitignore`，不会被提交到 Git
- 请勿将包含真实 API 密钥的配置文件分享给他人
- 如果不小心泄露了 API 密钥，请立即在火山引擎控制台重新生成

## 配置文件说明

- `planner/config.toml.example` - 配置文件模板（可以提交到 Git）
- `planner/config.toml` - 实际配置文件（包含敏感信息，已被忽略）
